# Bridge MSFS -> Arduino LCD

Esta pasta sera usada para o software opcional que le dados do MSFS 2024 no PC e envia frequencias para o Arduino pela Serial USB.

Protocolo esperado pelo firmware:

- ACT:124.850
- STBY:121.700

Cada comando deve terminar com \n.
